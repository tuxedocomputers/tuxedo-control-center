Power-saving CPU graphics processor (iGPU)
Exclusively uses the graphics processor integrated in the main processor. Suitable for power saving and low graphics performance requirements.
Please note: Monitors connected to the high-performance graphics processor do not work in this mode.
High performance graphics processor (dGPU)
Uses high-performance graphics processor for graphics-intensive tasks, but with higher energy consumption, even at low load and idle.
Hybrid graphics mode (on-demand)
The system prioritizes the energy-saving CPU graphics processor. The high-performance graphics processor can be added by the user if required. Combination of high performance and power saving.
